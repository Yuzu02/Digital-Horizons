## Prettier, ESLint & VSCode

In this article, we will learn how to setup Prettier with ESLint and VSCode.

## Step 1: Install Prettier and ESLint

First, we need to install Prettier and ESLint.

```bash
npm install --save-dev prettier eslint
```

## Step 2: Create ESLint configuration file

Next, we need to create an ESLint configuration file.

```bash
npx eslint --init
```

## Step 3: Create Prettier configuration file

Next, we need to create a Prettier configuration file.

```bash
echo {}> .prettierrc.json
```

## Step 4: Install Prettier and ESLint extensions in VSCode

Finally, we need to install Prettier and ESLint extensions in VSCode.

That's it! Now you have successfully setup Prettier with ESLint and VSCode.
