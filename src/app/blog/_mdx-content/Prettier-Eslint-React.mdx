<SubSeccion id="introduccion">Introducción</SubSeccion>

<Parrafo>
  En el mundo del desarrollo de software, mantener un código limpio y consistente es crucial. Prettier y ESLint son dos
  herramientas poderosas que pueden ayudarte a lograr esto. En este artículo, aprenderemos cómo configurar Prettier con ESLint y
  VSCode para mejorar tu flujo de trabajo de desarrollo.
</Parrafo>

<Alert type="info">
  Prettier es un formateador de código opinado, mientras que ESLint es una herramienta de linting que ayuda a identificar y
  reportar patrones en JavaScript. Cuando se usan juntos, pueden hacer maravillas para la calidad de tu código.
</Alert>

<SubSeccion id="step-1-install-prettier-and-eslint">Paso 1: Instalar Prettier y ESLint</SubSeccion>

<Parrafo>
  Lo primero que necesitamos hacer es instalar Prettier y ESLint en nuestro proyecto. Abre tu terminal y ejecuta el siguiente
  comando:
</Parrafo>

```bash
npm install --save-dev prettier eslint eslint-config-prettier eslint-plugin-prettier
```

<Parrafo>
  Este comando instala Prettier, ESLint, y dos paquetes adicionales que ayudan a que Prettier y ESLint trabajen juntos sin
  conflictos.
</Parrafo>

<SubSeccion id="step-2-create-eslint-configuration-file">Paso 2: Crear el archivo de configuración de ESLint</SubSeccion>

<Parrafo>
  A continuación, necesitamos crear un archivo de configuración de ESLint. Puedes hacerlo manualmente o usar el asistente de
  configuración de ESLint:
</Parrafo>

```bash
npx eslint --init
```

<Parrafo>
  Sigue las instrucciones en la terminal para configurar ESLint según tus preferencias. Una vez completado, tendrás un archivo
  \`.eslintrc.js\` en tu proyecto.
</Parrafo>

<SubSeccion id="step-3-create-prettier-configuration-file">Paso 3: Crear el archivo de configuración de Prettier</SubSeccion>

<Parrafo>
  Ahora, vamos a crear un archivo de configuración de Prettier. Crea un archivo llamado \`.prettierrc\` en la raíz de tu proyecto
  y añade las siguientes reglas:
</Parrafo>

```json
{
  "semi": true,
  "tabWidth": 2,
  "printWidth": 100,
  "singleQuote": true,
  "trailingComma": "es5",
  "jsxSingleQuote": true,
  "bracketSpacing": true
}
```

<Parrafo>Estas reglas son solo un ejemplo. Puedes ajustarlas según las preferencias de tu equipo o proyecto.</Parrafo>

<SubSeccion id="extensions-in-vscode">Paso 4: Instalar las extensiones de Prettier y ESLint en VSCode</SubSeccion>

<Parrafo>Para que todo funcione sin problemas en VSCode, necesitamos instalar las extensiones de Prettier y ESLint:</Parrafo>

<ol>
  <li>Abre VSCode</li>
  <li>Ve a la pestaña de extensiones (o presiona Ctrl+Shift+X)</li>
  <li>Busca "Prettier - Code formatter" e instálala</li>
  <li>Busca "ESLint" e instálala</li>
  <li>Reinicia VSCode</li>
</ol>

<Alert type="warning">
  Si no ves los cambios de formato de Prettier, asegúrate de que la opción "Format On Save" esté habilitada en VSCode. Puedes
  habilitarla yendo a "File" > "Preferences" > "Settings" y buscando "Format On Save".
</Alert>
